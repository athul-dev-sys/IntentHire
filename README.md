## Local Setup and Run Guide

### 1) Clone repository

```bash
git clone https://github.com/athul-dev-sys/IntentHire.git
cd IntentHire/frontend
```

### 2) Install dependencies

```bash
npm install
```

### 3) Configure environment variables

Create a `.env.local` file in `frontend` and add required variables used by the app.

### 4) Run frontend locally

```bash
npm run dev
```

Visit `http://localhost:3000`.

### 5) Run backend (if required for full flow)

From the `backend` directory, install dependencies and start backend services based on your existing backend setup.
